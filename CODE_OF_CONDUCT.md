# Code of Conduct

This project follows the [Wikimedia Code of Conduct](https://www.mediawiki.org/wiki/Code_of_Conduct).

## Summary

- Be respectful and welcoming
- Assume good faith
- Be collaborative
- When disagreeing, explain why
- Be concise

For more details, please see the full [Wikimedia Code of Conduct](https://www.mediawiki.org/wiki/Code_of_Conduct).
